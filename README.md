# MyPyrevitTools (MyTab)

此資料夾包含多個自訂的 pyRevit 工具，主要分為兩大面板：**GreenBuilding** 與 **Import**，分別提供綠建築日常節能指標檢核的開窗面積計算，以及從外部 CSV 檔案整批回寫圖紙變數的功能。

## 🛠️ 工具清單與功能說明

### 1. WindowOrientation (外殼等價開窗面積計算)
- **位置**: `MyTab` 標籤 -> `GreenBuilding` 面板
- **功能描述**:
  自動收集模型中的窗戶（Windows）與幕牆（Curtain Wall, DW），並依地區氣候、真北或專案北的方向，計算出對應的方位角及日射修正係數（fk）。最終計算出各窗戶的「外殼等價開窗面積（ΣAgi*fk*Ki）」，並匯出為 CSV 報表，方便進行台灣綠建築日常節能指標 (Req) 之檢核計算。
- **特色亮點**:
  - **內建氣候區數據**: 支援地區選擇（臺北市、臺中市、花蓮市、高雄市、臺東市），自動代入法規對應表 8 的日射修正係數（fk）。
  - **方位切換**: 可選擇使用「真北（True North）」或「專案北（Project North）」來精準判定窗戶的 16 方位。
  - **支援幕牆**: 計算對象可自訂勾選一般窗戶 (Windows)、幕牆 (CurtainWall DW) 或兩者皆選。
  - **自動輸出報表**: 會依據選定資料夾產出兩種報表：`Req_D1_rows.csv` (彙總分類報表) 與 `Req_instances.csv` (所有窗戶參數明細)。

### 2. Import from CSV (自 CSV 批次回寫圖框參數)
- **位置**: `MyTab` 標籤 -> `Import` 面板
- **功能描述**:
  將外部編輯好的 CSV 表單資料，尋找對應的依據鍵值（通常為圖號 Sheet Number），整批回寫同步至 Revit 專案的「圖紙 (Views Sheet)」以及「圖紙上的圖框實體 (Title Block)」內部參數。
- **特色亮點**:
  - **高相容性編碼讀取**: 支援多種 CSV 編碼格式（UTF-8, UTF-8-BOM, CP950 / Big5 免轉碼直接讀），不易發生亂碼。
  - **彈性對應建置**: 使用者可從視窗自行指定作為「比對鍵值」的 CSV 欄位名稱，以及想要寫回 Revit 之對應參數。
  - **智慧參數對應與快取**: 如果 CSV 的標題在 Revit 圖紙中沒有對應的一模一樣參數名稱，系統會跳出一次性的詢問視窗，使用者輸入對應的 Revit 參數名稱後，系統便會記憶該次對應並套用於其後所有該攔的資料更新。
  - **安全防護機制**: 即時統計成功與失敗的筆數與原因，並提供 Transaction 完整性，發生錯誤時自動回復 (Rollback) 以防止資料損壞。

## 📥 安裝與路徑設定
工具已經封裝為 `MyTools.extension` 擴充套件格式。可以透過 pyRevit 設定管理員 (Settings -> Custom Extension Directories) 將此資料夾的路徑加入，並重新載入 (Reload) 以在上方選單列顯示 `MyTab` 標籤。
