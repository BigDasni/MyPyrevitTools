# MyPyrevitTools (MyTab)

此資料夾包含多個自訂的 pyRevit 工具，主要分為四大面板：**GreenBuilding**、**Data**、**Sheet** 與 **Areas**，分別提供綠建築日常節能指標檢核的開窗面積計算、Revit 參數與外部 CSV 表單的雙向匯出與匯入回寫功能、圖紙圖框批次管理，以及空間邊界自動建立功能。

## 🛠️ 工具清單與功能說明

### 1. WindowOrientation (外殼等價開窗面積計算)
- **位置**: `MyTab` 標籤 -> `GreenBuilding` 面板
- **功能描述**:
  自動收集模型中的窗戶（Windows）與幕牆（Curtain Wall, DW），並依地區氣候、真北或專案北的方向，計算出對應的方位角及日射修正係數（fk）。最終計算出各窗戶的「外殼等價開窗面積（ΣAgi*fk*Ki）」，並匯出為 CSV 報表，方便進行台灣綠建築日常節能指標 (Req) 之檢核計算。
- **特色亮點**:
  - **內建氣候區數據**: 支援地區選擇（臺北市、臺中市、花蓮市、高雄市、臺東市），自動代入法規對應表 8 的日射修正係數（fk）。
  - **方位切換**: 可選擇使用「真北（True North）」或「專案北（Project North）」來精準判定窗戶的 16 方位。
  - **支援幕牆**: 計算對象可自訂勾選一般窗戶 (Windows)、幕牆 (CurtainWall DW) 或兩者皆選。
  - **自動輸出報表**: 會依據選定資料夾產出兩種報表：`Req_D1_rows.csv` (彙總分類報表) 與 `Req_instances.csv` (所有窗戶參數明細)。

### Data 面板工具 (CSV 資料匯出/匯入)

#### 2. Import Sheet CSV (自 CSV 批次回寫圖框參數)
- **位置**: `MyTab` 標籤 -> `Data` 面板 
- **功能描述**:
  將外部編輯好的 CSV 表單資料，透過指定的**獨特碼 (Unique Identifier)** 進行比對（通常為圖號 Sheet Number，但不強制，只要能 1 對 1 辨識圖紙的參數皆可），將整批資料回寫同步至 Revit 專案的「圖紙 (Views Sheet)」以及「圖紙上的圖框實體 (Title Block)」內部參數。不需要依賴 Revit 內建明細表匯出也能運作。
- **特色亮點**:
  - **自訂辨識鍵值**: 匯入表格不限定格式，使用者可彈性指定 CSV 中任何一欄作為辨識鍵。只要 Revit 內有對應且不重複的參數（例如自訂的 GUID 或圖紙編號），即可精準匹配圖紙與寫入。
  - **高相容性編碼讀取**: 支援多種 CSV 編碼格式（UTF-8, UTF-8-sig, CP950 / Big5 免轉碼直接讀），不易發生亂碼。
  - **彈性對應建置**: 使用者可從視窗自行指定作為「比對鍵值」的 CSV 欄位名稱，以及想要寫回 Revit 之對應參數。
  - **智慧參數對應與快取**: 如果 CSV 的標題在 Revit 圖紙中沒有對應的一模一樣參數名稱，系統會跳出一次性的詢問視窗，使用者輸入對應的 Revit 參數名稱後，系統便會記憶該次對應並套用於其後所有該攔的資料更新。
  - **安全防護機制**: 提供 Transaction 完整性，發生錯誤時自動回復 (Rollback) 以防止資料損壞。

#### 3. Export Room CSV (匯出房間參數至 CSV)
- **位置**: `MyTab` 標籤 -> `Data` 面板 -> `RoomTools` 群組
- **功能描述**:
  自動收集模型中已放置的房間 (Rooms)，讓使用者勾選欲匯出的參數 (如編號、名稱、面積、各項裝修材料等)，並將資料整理匯出為 UTF-8 (帶 BOM) 編碼的 CSV，方便使用 Excel 直接開啟編輯。
- **特色亮點**:
  - **自動轉換 Revit 單位**: 能夠抓取 Revit UI 上顯示的格式化數值（例如面積帶有單位），而非底層未轉換的英制數字。
  - **智慧預選**: 自動記憶並預選常用的房間參數欄位，加快操作流程。

#### 4. Import Room CSV (自 CSV 批次回寫房間參數)
- **位置**: `MyTab` 標籤 -> `Data` 面板 -> `RoomTools` 群組
- **功能描述**:
  讀取外部修改好的房間 CSV 檔案，使用指定的「對應鍵」（預設為房間編號）將修改後的資料（例如部門、名稱、裝修等）回寫至 Revit 的房間參數中。
- **特色亮點**:
  - **支援單位系統解析**: 自動透過 Revit 的 UnitUtils 單位系統解析帶有單位的字串，並正確轉換回 Revit 內部數值（不用擔心長度、面積等出現英制亂碼）。
  - **避免唯讀屬性錯誤**: 排除強制寫入唯讀屬性（如面積），並確保寫回安全。

### Sheet 面板工具 (圖紙/圖框管理)

#### 5. Change TitleBlock (批次更換圖框)
- **位置**: `MyTab` 標籤 -> `Sheet` 面板
- **功能描述**:
  批次更換所選圖紙的圖框類型 (TitleBlock Type)。選擇圖紙後，再選擇目標圖框類型，即可一次替換所有選中圖紙的 TitleBlock。使用 `ChangeTypeId()` 保留圖框位置與共用參數值。
- **特色亮點**:
  - **多選圖紙**: 使用 pyRevit 內建的圖紙選擇器，支援多選與篩選。
  - **執行前預覽**: 替換前會在 output 面板列出每張圖紙目前的圖框狀態。
  - **安全替換**: 使用 `ChangeTypeId()` 而非刪除再重建，保留位置與參數。
  - **智慧跳過**: 若圖紙已經使用目標圖框，則自動跳過不重複處理。

### Areas 面板工具 (空間邊界/尺度設定)

#### 6. Boundary create (自動建立房間邊界與標註)
- **位置**: `MyTab` 標籤 -> `Areas` 面板 
- **功能描述**:
  自動選取 Revit 內的房間（Rooms），並依據需求在指定的目標平面視圖（如 Area Plans 等）中建立對應的細部線（Detail Curves）、面積邊界線（Area Boundary Lines）以及長度標註（Length Dimensions），同時可自訂文字註記類型。
- **特色亮點**:
  - **多功能建立**: 提供勾選選項，自動轉換房間幾何邊界並繪製出對應的細部線或面積邊界線。
  - **跨視圖處理**: 支援選擇不同的 Level 與對應的 Target View（優先考量 Area Plans），直接將線條與標註建立在正確的視圖上。
  - **預設防呆與錯誤處理**: 全程透過 Revit API 精確抓取邊界（包含曲線等幾何），並內建重複標註的防護機制。

## 📥 安裝與路徑設定
工具已經封裝為 `MyTools.extension` 擴充套件格式。可以透過 pyRevit 設定管理員 (Settings -> Custom Extension Directories) 將此資料夾的路徑加入，並重新載入 (Reload) 以在上方選單列顯示 `MyTab` 標籤。
